## Component Breakdown

### 1. Hero Component (`CivieducationHero.tsx`)

```typescript
interface HeroProps {
  title: string;
  subtitle: string;
  ctaButtons: {
    label: string;
    href: string;
    variant: "primary" | "secondary" | "outline";
  }[];
  stats: {
    icon: string;
    value: string;
    label: string;
  }[];
}
```

**Design:**

- Background: Gradient with animated elements
- Typography: FoundersGrotesk for headings
- Stats: 3-column grid with icons

### 2. Learn Courses Component (`LearnCoursesSection.tsx`)

```typescript
interface Course {
  id: string;
  title: string;
  description: string;
  duration: string;
  lessons: number;
  href: string;
  status: "available" | "coming-soon";
}

interface LearnCoursesProps {
  courses: Course[];
}
```

**Design:**

- Cards: White background, rounded corners, subtle shadow
- Status badges: Green for available
- Hover effects: Scale up + shadow increase

### 3. Civic Topics Component (`CivicTopicsSection.tsx`)

```typescript
interface Topic {
  id: string;
  title: string;
  description: string;
  fullDescription: string;
  icon: string;
  href: string;
}

interface CivicTopicsProps {
  topics: Topic[];
}
```

**Design:**

- Grid: 3 columns (desktop), 2 (tablet), 1 (mobile)
- Cards: Icon + Title + Description + Full description on hover
- Arrow indicator for "Learn More"

### 4. Track Project Component (`TrackProjectSection.tsx`)

```typescript
interface FeaturedProject {
  title: string;
  county: string;
  budget: string;
  status: "completed" | "in-progress" | "allocated" | "stalled";
  href: string;
}

interface TrackProjectProps {
  featuredProjects: FeaturedProject[];
}
```

**Design:**

- Search bar: Large input with search button
- Project cards: Status color-coded border
- Status badges: Green (completed), Blue (in-progress), etc.

### 5. Analysis Preview Component (`AnalysisPreviewSection.tsx`)

```typescript
interface QuickStat {
  value: string;
  label: string;
}

interface AnalysisPreviewProps {
  stats: QuickStat[];
}
```

**Design:**

- Sector/County cards: Large icons with hover effects
- Stats: 4-column grid
- Background: Light accent color

### 6. CTA Component (`CTASection.tsx`)

```typescript
interface CTAProps {
  title: string;
  buttonText: string;
  buttonHref: string;
}
```

**Design:**

- Centered, large typography
- Gradient background
- Prominent CTA button
