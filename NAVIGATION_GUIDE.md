## Mobile Responsive Breakpoints

All pages use consistent Tailwind breakpoints:
- **smOnly**: Mobile phones (< 640px)
- **xm**: Small phones (640-768px)  
- **mdOnly**: Tablets (768-1024px)
- **Default**: Desktop (> 1024px)

Grid layouts:
- Desktop: 2-4 columns
- Tablet: 2 columns  
- Mobile: 1 column
